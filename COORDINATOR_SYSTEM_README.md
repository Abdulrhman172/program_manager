# نظام مسؤول البرنامج - Coordinator Management System

## نظرة عامة

نظام Flutter متكامل لإدارة ومتابعة بحوث التخرج من منظور مسؤول البرنامج. يوفر واجهة احترافية وسهلة الاستخدام بالكامل باللغة العربية لإدارة الطلاب والمشرفين والمراحل والبحوث والإشعارات.

## المميزات الرئيسية

### 1. لوحة التحكم الرئيسية (Dashboard)
- عرض إحصائيات شاملة عن البحوث والطلاب والمشرفين والمراحل المكتملة
- قسم الإشعارات الهامة مع تصنيفات مختلفة (تحذير، نجاح، معلومات)
- عرض المراحل الخمس للبحث مع حالة كل مرحلة
- قسم الأنشطة الأخيرة والإجراءات السريعة
- واجهة ديناميكية تدعم جميع أحجام الشاشات

### 2. إدارة الطلاب والمشرفين (Students Screen)
- جدول شامل لعرض بيانات الطلاب مع جميع المعلومات الأساسية
- بحث وتصفية متقدمة حسب الاسم والتخصص والبريد الإلكتروني
- عرض التخصص والمشرف الأكاديمي لكل طالب
- إمكانية التعديل والحذف والإضافة
- إحصائيات عن الطلاب النشطين والتخصصات المختلفة

### 3. إدارة المراحل (Stages Screen)
- عرض جميع مراحل البحث الخمس بتفاصيل كاملة
- تحديد حالة كل مرحلة (مكتملة، نشطة، قادمة)
- عرض تواريخ البداية والنهاية لكل مرحلة
- عدد المهام المرتبطة بكل مرحلة
- إمكانية التعديل والحذف والإضافة

### 4. اعتماد البحوث (Approval Screen)
- عرض البحوث قيد المراجعة مع جميع التفاصيل
- معلومات الطالب والمشرف والمرحلة
- أزرار اعتماد أو رفض البحث بسهولة
- عرض البحوث المعتمدة والمرفوضة في أقسام منفصلة
- إحصائيات الحالة المختلفة

### 5. الإشعارات (Notifications Screen)
- عرض جميع الإشعارات بأنواعها المختلفة
- تمييز الإشعارات المقروءة وغير المقروءة
- إمكانية وضع علامة كمقروء أو حذف
- تصنيفات ملونة حسب نوع الإشعار (تحذير، نجاح، معلومات)

### 6. الإعدادات (Settings Screen)
- تبويب الملف الشخصي: تعديل البيانات الشخصية والصورة
- تبويب الأمان: تغيير كلمة المرور وإدارة الجلسات
- تبويب الإشعارات: إدارة تفضيلات الإشعارات
- منطقة الخطر: تسجيل الخروج من جميع الأجهزة وحذف الحساب

- **الأزرق الأساسي:** #2563EB (للعناصر البارزة والأزرار)
- **الأخضر:** #10B981 (للحالات الناجحة)
- **الأحمر:** #EF4444 (للتنبيهات والأخطاء)
- **الأصفر:** #F59E0B (للتحذيرات)
- **الرمادي:** #F3F4F6 (للخلفيات الثانوية)

### الخطوط
- **الخط الأساسي:** Cairo و Tajawal (خطوط عربية حديثة)
- **حجم العناوين:** 24-32px (Bold)
- **حجم النصوص:** 14-18px (Regular)

### التخطيط
- **اتجاه RTL:** دعم كامل للغة العربية
- **Drawer:** على اليمين مع قائمة التنقل الرئيسية
- **AppBar:** يحتوي على البحث والإشعارات وبيانات المستخدم
- **Responsive:** يدعم جميع أحجام الشاشات
