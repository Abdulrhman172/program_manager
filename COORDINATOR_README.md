# نظام مسؤول البرنامج - Coordinator Dashboard System

## نظرة عامة

نظام متكامل لإدارة ومتابعة بحوث التخرج من منظور مسؤول البرنامج. يوفر واجهة احترافية وسهلة الاستخدام لإدارة الطلاب والمشرفين والمراحل والبحوث والإشعارات.

## المميزات الرئيسية

### 1. لوحة التحكم الرئيسية (Dashboard)
- عرض إحصائيات شاملة عن البحوث والطلاب والمشرفين والمراحل المكتملة
- قسم الإشعارات الهامة مع تصنيفات مختلفة (تحذير، نجاح، معلومات)
- عرض المراحل الخمس للبحث مع حالة كل مرحلة
- قسم الأنشطة الأخيرة والإجراءات السريعة

### 2. إدارة الطلاب والمشرفين
- جدول شامل لعرض بيانات الطلاب
- بحث وتصفية متقدمة
- عرض التخصص والمشرف الأكاديمي
- إمكانية التعديل والحذف
- إحصائيات عن الطلاب النشطين والتخصصات

### 3. إدارة المراحل
- عرض جميع مراحل البحث الخمس
- تحديد حالة كل مرحلة (مكتملة، نشطة، قادمة)
- عرض تواريخ البداية والنهاية
- عدد المهام لكل مرحلة
- إمكانية التعديل والحذف

### 4. اعتماد البحوث
- عرض البحوث قيد المراجعة
- معلومات الطالب والمشرف والمرحلة
- أزرار اعتماد أو رفض البحث
- عرض البحوث المعتمدة والمرفوضة
- إحصائيات الحالة

### 5. الإشعارات
- عرض جميع الإشعارات بأنواعها المختلفة
- تمييز الإشعارات المقروءة وغير المقروءة
- إمكانية وضع علامة كمقروء أو حذف
- تصنيفات ملونة حسب نوع الإشعار

### 6. الإعدادات
- تبويب الملف الشخصي: تعديل البيانات الشخصية
- تبويب الأمان: تغيير كلمة المرور وإدارة الجلسات
- تبويب الإشعارات: إدارة تفضيلات الإشعارات
- منطقة الخطر: تسجيل الخروج من جميع الأجهزة

## الهوية البصرية

### الألوان
- **الأزرق الأساسي:** #2563EB (للعناصر البارزة والأزرار)
- **الأخضر:** #10B981 (للحالات الناجحة)
- **الأحمر:** #EF4444 (للتنبيهات والأخطاء)
- **الأصفر:** #F59E0B (للتحذيرات)
- **الرمادي:** #F3F4F6 (للخلفيات الثانوية)

### الخطوط
- **الخط الأساسي:** Cairo و Tajawal (خطوط عربية حديثة)
- **حجم العناوين:** 24-32px (Bold)
- **حجم النصوص:** 14-18px (Regular)

### التخطيط
- **اتجاه RTL:** دعم كامل للغة العربية
- **Sidebar:** على اليمين مع قائمة التنقل الرئيسية
- **Header:** يحتوي على البحث والإشعارات وبيانات المستخدم
- **Responsive:** يدعم جميع أحجام الشاشات

## التقنيات المستخدمة

- **React 19:** مكتبة واجهات المستخدم
- **TypeScript:** للأمان والتطوير الموثوق
- **Tailwind CSS 4:** لتصميم الواجهات
- **shadcn/ui:** مكونات واجهة حديثة
- **Lucide React:** أيقونات عالية الجودة
- **Wouter:** للتنقل بين الصفحات
